## Weather Data Task


##Tech Stack:
+ AngularJS 1.5
+ Bootstrap
+ Karma-Jasmine
+ Typescript

##Getting started:
You need to have npm and bower installed in order to set up the application.<br>
Application can be accessed at http://localhost:3000/

```
$ git clone https://github.com/AlexN93/weather-data
$ bower install
$ npm run serve
```

##Testing:
```
$ npm test
```
